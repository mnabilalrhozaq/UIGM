```php
// Get current filter status
$statusFilter = $_GET['status'] ?? '';
$filterParam = $statusFilter ? '&status=' . $statusFilter : '';

// URL pagination dengan filter
base_url('/admin-pusat/manajemen-harga?page=' . $i . $filterParam)
```

Logic:
1. Ambil parameter status dari $_GET
2. Jika ada status, buat string '&status=aktif'
3. Jika tidak ada, string kosong ''
4. Append ke URL pagination

Contoh:
- Filter "Aktif": $filterParam = '&status=aktif'
- URL: /manajemen-harga?page=2&status=aktif

- Filter "Semua": $filterParam = ''
- URL: /manajemen-harga?page=2
