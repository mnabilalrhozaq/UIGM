üîß Step-by-Step Testing Guide

Step 1: Login as Admin Unit

  • Go to: http://localhost:8080/auth/login
  • Username: admin_unit
  • Password: password123
  • Verify successful login and redirect to dashboard

Step 2: Access Admin Unit Dashboard

  • URL: http://localhost:8080/admin-unit/dashboard
  • Verify all 6 UIGM categories are displayed
  • Check that progress bar shows 0% initially

Step 3: Test New Form Fields

  • Open any category form (e.g., WS - Waste)
  • Verify "Tanggal Input" field is present and defaults to today
  • Verify "Gedung/Lokasi" dropdown has building options
  • For WS category: Verify "Jenis Sampah" dropdown appears
  • Verify "Jumlah" and "Satuan" fields are present
  • Verify "Deskripsi" field is present

Step 4: Test Form Validation

  • Try saving with empty required fields - should show validation errors
  • Try entering negative number in "Jumlah" - should show error
  • Try entering description less than 10 characters - should show error
  • Fill all required fields correctly - should save successfully

Step 5: Test Data Saving

  • Fill out complete form with valid data
  • Click "Simpan" button
  • Verify success message appears
  • Verify progress bar updates
  • Verify category status changes to "Lengkap"

Step 6: Test Complete Workflow

  • Fill out all 6 categories with complete data
  • Verify progress reaches 100%
  • Verify "Kirim ke Admin Pusat" button becomes enabled
  • Click "Kirim ke Admin Pusat"
  • Verify data is sent successfully

Step 7: Test Admin Pusat Review

  • Logout and login as Admin Pusat
  • Username: admin_pusat, Password: password123
  • Go to monitoring page and find submitted data
  • Click review and verify all new fields are displayed
  • Verify data can be reviewed and approved
